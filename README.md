
# BookManager-SpringBoot
A modern Spring Boot book management system featuring Apple‑style glassmorphism UI, file uploads, image previews, and full CRUD operations.  Educational Spring Boot BookManager with modern UI. Security hardening planned for future releases. 
🔐 Security Update: Hashing Algorithm Improved
The project previously used an MD5-based hashing utility.
MD5 is considered insecure due to collision vulnerabilities, so the hashing algorithm has been upgraded to SHA‑256 for improved security and reliability.

Please note:
The class name is still MD5Hash for now, but the internal implementation has been fully migrated to SHA‑256.
The class will be renamed in a future update to reflect the new hashing standard.

This change enhances the overall security of the application and aligns it with modern cryptographic best practices.
